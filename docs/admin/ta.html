    作業系統課程 助教工作指引 (TA Guide)

    網站建構

    1. 安裝 quarto
    2. 在此 repository 根目錄打 quarto preview
    3. 網站的 markdown 會被轉換成 html 並顯示在瀏覽器

    網站裡面有小考：

    1. 根據這個來安裝R
    2. 在 cmdline 進入 R
    3. install.packages("exams", dependencies = TRUE)
    4. install.packages("exams2forms", dependencies = TRUE)

    小考的 syntax

    macOS Sequoia 15.5 arm64 installation guide:

    1. Install homebrew: /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    2. Install quarto: brew install --cask quarto
    3. Install R: brew install r
    4. Install R dependencies:
    $ R

    工作分配

    1. 8/27 or 8/28 (開學前一週) 助教會議
    2. 預計收90位學生，每組三人，共15組。每一位助教負責5組

    每週職責

    1. 鼓勵學生學習

    • 影片發布後，在 Discord 發貼文鼓勵同學看影片並且分享補充資料 「這週有讓你”燒蛋一下”的地方嗎？」
    • 幫助學生把模糊的疑問轉成具體問題。

    2. 小組實作的教練

    • 週一小組實作時，協助觀察小組討論狀況、給予提示。
    • 協助管理搶答活動的記分與氣氛。

    3. 問題探索 篩選

    • 蒐集學生在 Discord po 的問題與補充資料，篩選出有價值的問題推薦給老師在週三課堂 highlight (shio time)。

    4. 課堂 Lab & Quiz 驗證

    • 每週教授會出5題的小考題目，以及 Lab，請幫忙 go through 一遍並且找出可能的 bug
    • 先把課堂 lab 做過一遍以熟悉內容

    5. 每週線上 TA Hour

    • 每週安排一次 TA線上答疑時段 (Office Hour)，讓學生可以進來討論Lab問題、探索專案的想法，或是問任何影片看不懂的地方。
    • TA Hour 不需要每次都準備課程內容，重點是「在場」讓學生知道有求救窗口。
    • 可以用 Discord Voice Channel

    6. 期中考 & 期末考出題、改題協助

    • 協助老師設計考題題庫（選擇題 / 概念題），聚焦在學生容易誤解的知識點。
    • 協助檢查考題是否明確，避免模糊題意。
    • 改考卷
